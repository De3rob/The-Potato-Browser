## The Potato Browser 
Install browser here:
[The Potato Browser releases page](https://github.com/De3rob/The-Potato-Browser/releases/tag/v0.0.1).


A lightweight, modern web browser built with Electron.

### **Features**

-  Tabbed browsing with favicon indicators
-  Dark theme UI
-  Switchable search engines (Google, DuckDuckGo, Bing, Brave)
-  Navigation buttons (Back, Forward, Reload, Home)
-  Settings modal with persistent preferences

### **Installation**

Download and run PotatoBrowser-Setup.exe to install.
Or you can install application as a zip file and run PotatoBrowser.exe




### **What's Inside**

- Windows x64 installer
- ~211 MB standalone application
- No external dependencies needed
